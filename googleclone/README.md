# Google Clone

This is a simple clone of the Google homepage created using HTML and CSS.
![ss](https://github.com/Researcher0619/Patika-Front-End-/assets/102052449/9a6f5b0d-3771-41d4-a26a-703ff3c9d89e)



## Description

This project aims to replicate the basic layout and functionality of the Google homepage using only HTML and CSS. It includes a search bar, Google logo, and buttons styled similar to the original Google homepage.

## Features

- Responsive design: The page is designed to adapt to different screen sizes.
- Search functionality: Users can enter search queries in the search bar.
- Google logo: A replica of the Google logo is displayed at the top of the page.
- Buttons: Styled buttons for Google Search and "I'm Feeling Lucky" are included.
